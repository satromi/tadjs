# プラグイン利用ガイド

BTRON Desktopでは、プラグインを追加することで機能を拡張できます。

---

## 1. 標準プラグイン一覧

### 原紙タイプ（原紙箱に表示）

| プラグイン | 説明 |
|-----------|------|
| 基本文章編集 | TADファイル用リッチテキストエディタ |
| 基本図形編集 | 図形を描画・編集するエディタ |
| 基本表計算 | スプレッドシート編集 |
| 予定表 | カレンダー表示 |
| URL仮身 | 実身URLでブラウザ起動 |

### 小物タイプ（アクセサリメニューから起動）

| プラグイン | 説明 |
|-----------|------|
| システム設定 | システム全体の設定 |
| 個人設定 | ユーザー個別の設定 |

### ビューア・ユーティリティタイプ

| プラグイン | 説明 |
|-----------|------|
| 仮身一覧 | 実身内の仮身を一覧表示 |
| TADビューア | TADファイルの表示 |
| ファイル取り込み | 外部ファイルのインポート |
| 書庫展開 | BPK書庫ファイルの展開 |
| ごみ箱 | 削除された実身の管理 |

## 2. プラグインの追加

### 手順

1. **プラグインフォルダを配置**

   `plugins/` ディレクトリに新しいプラグインフォルダを配置します。

   ```text
   plugins/
   ├── basic-text-editor/
   ├── virtual-object-list/
   └── my-new-plugin/  ← ここに配置
   ```

2. **アプリケーションを再起動**

   BTRON Desktopを再起動すると、自動的に新しいプラグインが認識されます。

### 配布パッケージへの追加

配布版（`dist/TADjs Desktop-win32-x64/`）に追加する場合：

```
dist/TADjs Desktop-win32-x64/
└── resources/
    └── app/
        └── plugins/
            └── my-new-plugin/  ← ここに配置
```

---

## 3. プラグインの無効化・削除

### 無効化

プラグインを一時的に無効にしたい場合：

- **フォルダ名の先頭にアンダースコアを追加**

  例: `my-plugin` → `_my-plugin`

### 削除

プラグインを完全に削除する場合：

- `plugins/` ディレクトリからフォルダを削除

---

## 4. トラブルシューティング

### プラグインが表示されない

1. プラグインフォルダが `plugins/` ディレクトリ直下にあるか確認
2. `plugin.json` ファイルが存在するか確認
3. アプリケーションを再起動

### プラグインが起動しない

1. 開発者ツール（`Ctrl+Shift+I`）でコンソールを確認
2. `plugin.json` のJSON形式が正しいか確認
3. `main` で指定されたHTMLファイルが存在するか確認
